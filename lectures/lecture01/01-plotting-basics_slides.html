Why Plotting Matters in Physics üìä

As physicists, we need to visualize our data and calculations.

A good plot can:

  • Reveal patterns and relationships
  • Make predictions visible
  • Communicate results effectively
  • Help us understand complex phenomena

Python‚Äôs Matplotlib library is the standard tool for creating scientific plots.

Getting Started with Matplotlib

Matplotlib is not built into Python ‚Äî we need to import it:

import matplotlib.pyplot as plt
import numpy as np

Convention: We import matplotlib.pyplot as plt

This is the standard shorthand used everywhere in Python!

Think of plt as your plotting toolkit ‚Äî all the functions you need are accessed through it.

Your First Plot

The most basic plotting command is:

plt.plot(x, y)
plt.show()

Let‚Äôs plot a sine wave:

# Create x values
x = np.linspace(0, 4*np.pi, 100)

# Calculate y values
y = np.sin(x)

# Create the plot
plt.figure(figsize=(6, 4))
plt.plot(x, y)
plt.show()

Understanding the Code

x = np.linspace(0, 4*np.pi, 100)

Creates 100 evenly-spaced points from 0 to 4œÄ

plt.figure(figsize=(6, 4))

Creates a new figure that‚Äôs 6√ó4 inches

plt.plot(x, y)

Plots y versus x

plt.show()

Displays the plot

Adding Labels ‚Äî Essential!

A plot without labels is useless! Always add:

  • x-axis label: What‚Äôs on the horizontal axis?
  • y-axis label: What‚Äôs on the vertical axis?
  • Title: What does this plot show?
plt.figure(figsize=(6, 4))
plt.plot(x, y)
plt.xlabel('Angle (radians)')
plt.ylabel('sin(x)')
plt.title('Sine Function')
plt.show()

Using LaTeX in Labels

You can use mathematical notation in labels with LaTeX syntax:

plt.xlabel(r'$\theta$ (radians)')  # Greek letters
plt.ylabel(r'$\sin(\theta)$')      # Math expressions

The r before the string means ‚Äúraw string‚Äù ‚Äî it treats backslashes literally.

Examples:

Code Result

Physics Example: Projectile Motion üéØ

Let‚Äôs plot something more physics-y! A ball launched at 45¬∞:

# Physics parameters
v0 = 20.0       # initial velocity (m/s)
angle = 45.0    # launch angle (degrees)
g = 9.81        # gravity (m/s¬≤)

# Convert angle to radians
theta = np.radians(angle)

# Time array
t_max = 2 * v0 * np.sin(theta) / g
t = np.linspace(0, t_max, 100)

# Calculate trajectory
x = v0 * np.cos(theta) * t
y = v0 * np.sin(theta) * t - 0.5 * g * t**2

Projectile Motion: The Plot

plt.figure(figsize=(6, 4))
plt.plot(x, y)
plt.xlabel('Horizontal Distance (m)')
plt.ylabel('Height (m)')
plt.title('Projectile Motion (45¬∞ Launch)')
plt.grid(True, alpha=0.3)
plt.show()

What‚Äôs new?

plt.grid(True, alpha=0.3) ‚Äî Adds a grid (alpha=0.3 makes it semi-transparent)

Physics equations turned into beautiful visualization!

Customizing Your Plot: Line Styles

plt.plot(x, y, '-')    # solid line
plt.plot(x, y, '--')   # dashed line
plt.plot(x, y, ':')    # dotted line

Combine with colors:

plt.plot(x, y, 'b--')  # blue dashed line
plt.plot(x, y, 'g:')   # green dotted line

Customizing Your Plot: Markers

For discrete data points:

plt.plot(x, y, 'o')    # circles
plt.plot(x, y, 's')    # squares
plt.plot(x, y, '^')    # triangles
plt.plot(x, y, '*')    # stars
plt.plot(x, y, '+')    # plus signs

Combine everything:

plt.plot(x, y, 'ro')    # red circles
plt.plot(x, y, 'b-o')   # blue line with circles
plt.plot(x, y, 'g--s')  # green dashed with squares

Example: Sine and Cosine

x = np.linspace(0, 2*np.pi, 50)

plt.figure(figsize=(6, 4))
plt.plot(x, np.sin(x), 'r-', linewidth=2)
plt.plot(x, np.cos(x), 'b--', linewidth=2)
plt.xlabel('x')
plt.ylabel('y')
plt.title('Sine and Cosine')
plt.grid(True, alpha=0.3)
plt.show()

New parameter: linewidth=2 makes the line thicker

Adding a Legend

When you have multiple lines, use a legend to identify them:

plt.figure(figsize=(6, 4))
plt.plot(x, np.sin(x), 'r-', linewidth=2, label='sin(x)')
plt.plot(x, np.cos(x), 'b--', linewidth=2, label='cos(x)')
plt.xlabel('x')
plt.ylabel('y')
plt.title('Trigonometric Functions')
plt.legend(loc='upper right')
plt.grid(True, alpha=0.3)
plt.show()

Key: Add label='...' to each plot, then call plt.legend()

Legend Locations

Common locations for plt.legend(loc=...):

'lower left' Bottom left corner

'best' Automatic (avoids data)

Physics Example: Damped Oscillator üåä

Let‚Äôs visualize different damping regimes:

omega0 = 2.0  # natural frequency
t = np.linspace(0, 10, 500)

# Different damping coefficients
gamma1 = 0.2   # underdamped
gamma2 = 2.0   # critically damped
gamma3 = 4.0   # overdamped

# Calculate responses
y1 = np.exp(-gamma1*t) * np.cos(omega0*t)
y2 = np.exp(-gamma2*t)
y3 = np.exp(-gamma3*t)

Damped Oscillator: The Plot

plt.figure(figsize=(8, 5))
plt.plot(t, y1, 'b-', linewidth=2, label=r'Underdamped ($\gamma=0.2$)')
plt.plot(t, y2, 'r-', linewidth=2, label=r'Critical ($\gamma=2.0$)')
plt.plot(t, y3, 'g-', linewidth=2, label=r'Overdamped ($\gamma=4.0$)')
plt.xlabel('Time (s)')
plt.ylabel('Amplitude')
plt.title('Damped Harmonic Oscillator')
plt.legend(loc='upper right')
plt.grid(True, alpha=0.3)
plt.show()

Notice: LaTeX in labels: r'Underdamped ($\gamma=0.2$)'

Plotting Data Points (Not Just Lines)

Sometimes you want to show discrete experimental data:

# Simulated experimental data
x_data = np.array([1, 2, 3, 4, 5, 6, 7, 8])
y_data = np.array([2.1, 3.9, 6.2, 7.8, 10.1, 11.9, 14.2, 15.8])

# Theoretical line
x_theory = np.linspace(0, 9, 100)
y_theory = 2 * x_theory

plt.figure(figsize=(6, 4))
plt.plot(x_data, y_data, 'ro', markersize=8, label='Experimental Data')
plt.plot(x_theory, y_theory, 'b-', linewidth=2, label='Theory: y = 2x')
plt.legend(loc='upper left')
plt.show()

Markers vs. Lines

Code Result
'ro' Red circles only (no line)
'r-' Red line only (no markers)
'r-o' Red line with circles
'r--o' Red dashed line with circles

For experimental data: Usually use markers only ('ro')

For theoretical curves: Usually use lines only ('b-')

Saving Your Plots üíæ

To save a plot to a file:

plt.savefig('my_plot.png')           # PNG format
plt.savefig('my_plot.pdf')           # PDF format (best for papers!)
plt.savefig('my_plot.png', dpi=300)  # High resolution PNG

‚ö†Ô∏è Important: Save BEFORE plt.show()!

plt.figure(figsize=(6, 4))
plt.plot(x, y)
plt.xlabel('x')
plt.ylabel('y')
plt.savefig('my_figure.pdf')  # Save first!
plt.show()                     # Then display

File Formats for Plots

Format Best For
PDF Papers, LaTeX documents, publications
PNG Presentations, web, general use
SVG Vector graphics, editing in Illustrator
EPS Some journals still require this

Pro tip: Always save as PDF for scientific work ‚Äî it‚Äôs resolution-independent!

üéØ Practice Exercise 3: Compare Trajectories

Plot projectile trajectories for launch angles of 30¬∞, 45¬∞, and 60¬∞ on the same plot.

Requirements:

  • Use \(v_0 = 20\) m/s for all
  • Add a legend
  • Use different colors for each angle

Hint: Use a loop!

for angle in [30, 45, 60]:
    # Calculate and plot each trajectory

Solution: Compare Trajectories

v0 = 20.0
g = 9.81
angles = [30, 45, 60]
colors = ['r', 'b', 'g']

plt.figure(figsize=(8, 5))
for angle, color in zip(angles, colors):
    theta = np.radians(angle)
    t_max = 2 * v0 * np.sin(theta) / g
    t = np.linspace(0, t_max, 100)
    x = v0 * np.cos(theta) * t
    y = v0 * np.sin(theta) * t - 0.5 * g * t**2
    plt.plot(x, y, color=color, linewidth=2, label=f'{angle}¬∞')

plt.xlabel('Horizontal Distance (m)')
plt.ylabel('Height (m)')
plt.title('Projectile Trajectories at Different Angles')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Common Mistakes to Avoid ‚ö†Ô∏è

  1. Forgetting plt.show() ‚Äî Your plot won‚Äôt display!
  2. No labels ‚Äî Always label axes and add a title
  3. Saving after showing ‚Äî Use plt.savefig() BEFORE plt.show()
  4. Too many points ‚Äî 100-500 points is usually enough for smooth curves
  5. Ugly default sizes ‚Äî Always set figsize for better proportions
