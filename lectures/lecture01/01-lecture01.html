Getting Started with Jupyter Notebooks

Why Jupyter? üöÄ

As physicists, we need to:

  • Calculate results from equations
  • Visualize data and theoretical predictions
  • Document our work with equations and explanations
  • Share reproducible results with colleagues

Jupyter Notebooks let us do all of this in one place!

What You‚Äôll Create Today

By the end of this Lecture, you‚Äôll generate plots like this:

All from physics equations you already know!

What is a Jupyter Notebook?

A Jupyter Notebook is a web-based interactive document that combines:

  • Code (Python) that you can run
  • Results (numbers, plots, animations)
  • Documentation (text, equations in LaTeX)

Think of it as a digital lab notebook for computational physics.

Key Advantage

Unlike traditional programming where you write everything first and then run it, Jupyter lets you experiment interactively - run small pieces of code, see results immediately, and build up your solution step by step.

Two Types of Cells

Jupyter notebooks consist of cells - individual blocks that contain either code or text.

Code Cells (Default)

  • Write Python code here
  • Run to see results
  • Identified by [ ]: on the left

Markdown Cells

  • Write notes, explanations, equations
  • Use for documentation
  • Change a cell to Markdown: Click cell, then press M key

Your First Calculation üéØ

Let‚Äôs calculate gravitational potential energy!

Physics

\[E_{\text{pot}} = mgh\]

Code

Type this in your first cell:

# Gravitational potential energy
m = 2.0    # mass in kg
g = 9.81   # gravity in m/s¬≤
h = 10.0   # height in m

E_pot = m * g * h

print(f"Potential energy: {E_pot} J")

Run It!

  • Click the ‚ñ∂ button (top of cell), OR
  • Press Shift + Enter (run and move to next cell), OR
  • Press Ctrl/Cmd + Enter (run and stay)

You should see: Potential energy: 196.2 J

Try It: Multi-Step Calculation

Let‚Äôs calculate the time for a ball to fall from height h.

Physics: \(h = \frac{1}{2}gt^2\) ‚Üí \(t = \sqrt{\frac{2h}{g}}\)

Your Task:

  1. Create a new cell below (press B)
  2. Type this code:
import numpy as np  # We need square root

h = 10.0   # height in m
g = 9.81   # gravity in m/s¬≤

t = np.sqrt(2 * h / g)

print(f"Fall time: {t:.2f} seconds")
  1. Run it! (Shift + Enter)

Adding Documentation with Markdown

Good physics notebooks explain what you‚Äôre doing!

Create a Markdown Cell

  1. Insert a new cell above your code (press Esc, then A)
  2. Change it to Markdown (press M)
  3. Type:
## Free Fall Calculation

We calculate the time for an object to fall from height $h = 10$ m.

The equation is:
$$t = \sqrt{\frac{2h}{g}}$$

where $g = 9.81 \, \text{m/s}^2$ is gravitational acceleration.
  1. Run the cell (Shift + Enter) to render it
LaTeX Equations
  • Inline equation: $E = mc^2$ ‚Üí \(E = mc^2\)
  • Display equation: $$E = mc^2$$ ‚Üí \[E = mc^2\]

Now let‚Äôs visualize physics! This is where Jupyter really shines - you can see your calculations come to life.

Plotting a Falling Ball

Let‚Äôs plot the height of our falling ball over time. Don‚Äôt worry about understanding every detail - just see how easy it is to create beautiful graphs!

Create a new cell and type:

import matplotlib.pyplot as plt
import numpy as np

# Calculate fall time
h = 10.0
g = 9.81
t_fall = np.sqrt(2 * h / g)

# Create time points from 0 to landing
t = np.linspace(0, t_fall, 50)

# Calculate height at each time: h = h0 - (1/2)gt¬≤
height = h - 0.5 * g * t**2

# Create the plot
plt.plot(t, height, 'b-', linewidth=2)
plt.xlabel('Time (s)')
plt.ylabel('Height (m)')
plt.title('Ball Falling from 10m')
plt.grid(True, alpha=0.3)
plt.show()

Run it! You just created your first physics visualization! üéâ

What Just Happened?
  • matplotlib.pyplot is Python‚Äôs plotting library
  • np.linspace(0, t_fall, 50) creates 50 evenly-spaced time points
  • plt.plot() draws the curve
  • The labels and grid make it look professional

Don‚Äôt memorize this yet - we‚Äôll cover all the details in Lecture 4. For now, just enjoy seeing your calculation as a graph!

Challenge: Plot Velocity

The velocity increases as the ball falls: \(v = gt\)

Try modifying the code to plot velocity vs time instead!

Solution 
# Calculate velocity at each time point
velocity = g * t

plt.plot(t, velocity, 'r-', linewidth=2)
plt.xlabel('Time (s)')
plt.ylabel('Velocity (m/s)')
plt.title('Velocity of Falling Ball')
plt.grid(True, alpha=0.3)
plt.show()

Notice how velocity increases linearly - exactly what the equation predicts!

Advanced Topics (Optional)

What is a Kernel?

The kernel is the computational engine that runs your code. Think of it as Python running in the background.

Kernel indicator (top-right):

  • ‚ö™ Idle: Ready to run code
  • ‚ö´ Busy: Currently executing

Kernel menu operations:

  • Restart Kernel: Clears all variables, starts fresh
  • Interrupt Kernel: Stops long-running code

You‚Äôll need to restart if:

  • Variables get messy
  • Code is stuck in an infinite loop (interrupt first, then restart)
  • You want a clean slate
