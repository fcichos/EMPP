The Schr√∂dinger Equation

The stationary Schr√∂dinger equation in one dimension:

\[-\frac{\hbar^2}{2m}\frac{d^2 \Psi(x)}{dx^2} + V(x)\Psi(x) = E\Psi(x)\]

Don‚Äôt panic! Let‚Äôs break this down mathematically:

Symbol Mathematical Role Physical Meaning (optional)
\(\Psi(x)\) Unknown function Wave function
\(\frac{d^2 \Psi}{dx^2}\) Second derivative Kinetic energy term
\(V(x)\) Given function Potential energy
\(E\) Unknown constant Eigenvalue (energy)

What Kind of Problem Is This?

This is an eigenvalue problem!

Compare to linear algebra:

\[A\vec{x} = \lambda\vec{x}\]

Meaning: Find special vectors \(\vec{x}\) that, when multiplied by matrix \(A\), just get scaled by a constant \(\lambda\) (not rotated or changed in direction).

The Schr√∂dinger equation:

\[\underbrace{\left(-\frac{\hbar^2}{2m}\frac{d^2}{dx^2} + V(x)\right)}_{\text{Operator } \hat{H}} \Psi(x) = E\Psi(x)\]

We‚Äôre looking for:

  • Eigenfunctions \(\Psi(x)\): special functions unchanged (up to a constant) by the operator
  • Eigenvalues \(E\): the constants (allowed energies)

The Strategy: Convert to Matrix Problem

  1. Discretize the spatial domain \(x \to x_1, x_2, \ldots, x_N\)
  2. Approximate the second derivative as a matrix
  3. Build the potential as a diagonal matrix
  4. Combine into the Hamiltonian matrix \(\mathbf{H}\)
  5. Solve \(\mathbf{H}\vec{\Psi} = E\vec{\Psi}\) with scipy

This is exactly like what we did for ODEs, but now we solve an eigenvalue problem instead of time-stepping!

Step 1: Discretize the Domain

Instead of \(\Psi(x)\) for all \(x\), evaluate at discrete points:

\[x_1, x_2, x_3, \ldots, x_N \quad \text{with spacing } \delta x\]

The function becomes a vector:

\[\Psi(x) \to \vec{\Psi} = \begin{pmatrix} \Psi(x_1) \\ \Psi(x_2) \\ \vdots \\ \Psi(x_N) \end{pmatrix}\]

This is familiar! Same discretization we used for ODEs and PDEs.

Step 2: The Second Derivative as a Matrix

Remember the finite difference formula:

\[\frac{d^2\Psi}{dx^2}\bigg|_{x_i} \approx \frac{\Psi(x_{i+1}) - 2\Psi(x_i) + \Psi(x_{i-1})}{\delta x^2}\]

Key insight: This combines values at neighboring points with weights!

  • Weight \(+1\) for \(\Psi(x_{i+1})\)
  • Weight \(-2\) for \(\Psi(x_i)\)
  • Weight \(+1\) for \(\Psi(x_{i-1})\)

The Second Derivative Matrix

\[\frac{d^2}{dx^2}\vec{\Psi} \approx \frac{1}{\delta x^2} \begin{pmatrix} -2 & 1 & 0 & 0 & \cdots & 0\\ 1 & -2 & 1 & 0 & \cdots & 0\\ 0 & 1 & -2 & 1 & \cdots & 0\\ \vdots & & \ddots & \ddots & \ddots & \vdots\\ 0 & \cdots & 0 & 1 & -2 & 1\\ 0 & \cdots & 0 & 0 & 1 & -2\\ \end{pmatrix} \begin{pmatrix} \Psi_1\\ \Psi_2\\ \Psi_3\\ \vdots\\ \Psi_{N-1}\\ \Psi_N \end{pmatrix}\]

This is a tridiagonal matrix:

  • Main diagonal: \(-2\)
  • Off-diagonals: \(+1\)

Building the Matrix in Python

from scipy.sparse import diags
import numpy as np

N = 1001  # number of grid points
dx = 0.01  # grid spacing

# Create the tridiagonal matrix
# diags([values], [positions]) where 0=main, ¬±1=off-diagonals
D2 = diags([-2, 1, 1], [0, -1, 1], shape=(N, N)) / dx**2

Why sparse? The matrix is mostly zeros ‚Äî sparse storage is efficient!

Boundary Conditions

The matrix implicitly assumes:

\[\Psi(x_0) = 0 \quad \text{and} \quad \Psi(x_{N+1}) = 0\]

These are Dirichlet boundary conditions ‚Äî the wavefunction vanishes at the boundaries.

Physical interpretation: The particle is confined to a region and cannot escape to infinity.

Step 3: The Potential Energy Matrix

The term \(V(x)\Psi(x)\) is simpler ‚Äî just multiplication at each point!

In matrix form, this is a diagonal matrix:

\[\mathbf{V} = \begin{pmatrix} V(x_1) & 0 & \cdots & 0\\ 0 & V(x_2) & \cdots & 0\\ \vdots & & \ddots & \vdots\\ 0 & \cdots & 0 & V(x_N)\\ \end{pmatrix}\]

Building the Potential Matrix

# Example: Particle in a box (rectangular well)
V_0 = 20  # barrier height in eV
d = 1e-9  # box width

# Potential is 0 inside the box, V_0 outside
V_vec = np.zeros(N)
V_vec[np.abs(x) > d/2] = V_0

# Create diagonal matrix
V = diags([V_vec], [0])

The potential defines the problem! Different \(V(x)\) = different physics.

Step 4: Assemble the Hamiltonian

Combining kinetic and potential energy:

\[\mathbf{H} = -\frac{\hbar^2}{2m}\mathbf{D}_2 + \mathbf{V}\]

hbar = 1.055e-34  # J¬∑s
m_e = 9.109e-31   # kg (electron mass)

# Kinetic energy matrix
T = -hbar**2 * D2 / (2 * m_e)

# Total Hamiltonian
H = T + V

That‚Äôs it! We‚Äôve converted the differential equation to a matrix.

Step 5: Solve the Eigenvalue Problem

Now we solve \(\mathbf{H}\vec{\Psi} = E\vec{\Psi}\):

from scipy.sparse.linalg import eigsh

# Find the n smallest eigenvalues and eigenvectors
n = 10  # number of states to find
eigenvalues, eigenvectors = eigsh(H, k=n, which='SM')

What we get:

  • eigenvalues: Array of energies \(E_1, E_2, \ldots, E_n\)
  • eigenvectors: Matrix where column \(k\) is \(\vec{\Psi}_k\)

Understanding the Output

# eigenvalues are the allowed energies
print("First 5 energy levels (eV):")
for i in range(5):
    print(f"  E_{i+1} = {eigenvalues[i] * 6.242e18:.3f} eV")

# eigenvectors are the wavefunctions
psi_1 = eigenvectors[:, 0]  # ground state
psi_2 = eigenvectors[:, 1]  # first excited state

Key result: Only certain discrete energies are allowed ‚Äî these are the eigenvalues!

Visualizing the Solutions

Each curve shows \(|\Psi(x)|^2\) (probability density) at its energy level.

The Recipe: Summary

  1. Define the grid: x = np.linspace(-L/2, L/2, N)
  2. Build kinetic matrix: T = -hbar¬≤/(2m) √ó diags([-2,1,1], [0,-1,1]) / dx¬≤
  3. Build potential matrix: V = diags([V_vec], [0])
  4. Combine: H = T + V
  5. Solve: eigenvalues, eigenvectors = eigsh(H, k=n, which='SM')
  6. Plot and interpret!

Why Does This Work?

Continuous Problem Discrete Problem
Differential operator \(\hat{H}\) Matrix \(\mathbf{H}\)
Eigenfunction \(\Psi(x)\) Eigenvector \(\vec{\Psi}\)
Eigenvalue \(E\) Eigenvalue \(E\)
Infinite dimensions \(N\) dimensions

As \(N \to \infty\) and \(\delta x \to 0\), the discrete solution approaches the exact solution!

Solution: Building D2

import numpy as np

N = 101
x = np.linspace(0, 1, N)
dx = x[1] - x[0]
f = x**2

# Method 1: Using np.diag
main = np.diag(np.full(N, -2))
upper = np.diag(np.ones(N-1), 1)
lower = np.diag(np.ones(N-1), -1)
D2 = (main + upper + lower) / dx**2

# Apply to f
f_second = D2 @ f
print(f"f''(0.5) = {f_second[N//2]:.4f}")  # Should be ‚âà 2

Comparing with Analytical Solutions

For a particle in an infinite box of width \(L\):

\[E_n = \frac{n^2 \pi^2 \hbar^2}{2mL^2}, \quad n = 1, 2, 3, \ldots\]

Key observation: \(E_n \propto n^2\)

Our numerical solution should match this scaling!

Energy Level Scaling

# Plot eigenvalues vs quantum number

# Compare with n¬≤ scaling

plt.xlabel('Quantum number n')
plt.ylabel('Energy')
plt.legend()

The numerical results confirm \(E_n \propto n^2\)!
