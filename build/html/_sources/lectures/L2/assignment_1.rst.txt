Übungszettel 1
==============

Als erste Übung melden Sie sich bei unserem JupyterHub Server an. Wir haben Ihnen eine IP-Adresse sowie Login und Passwort zur Verfügung gestellt. Machen Sie sich dort mit der Jupyter-System vertraut. Sie entspricht der JupyterLab-Oberfläche. Um zur Aufgabe zu gelangen, müssen Sie die **URL** in der Adresszeile des Browsers ändern. 

Die erste Übung bezieht sich auf

1. Markdown
2. LaTeX und Markdown
3. Strings
4. Lists
5. Dictionaries

Bitte vergessen Sie nicht, Ihre Übung am Ende einzureichen. Sie erhalten eine Rückmeldung wenn alles korrigiert ist. Wir werden im Seminar 3.11.2023 über die Lösungen sprechen.
